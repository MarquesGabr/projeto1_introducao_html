Nessa aula foi apresentada a estrutura básica de uma página HTML, que consiste em:

  • Head - cabeçalho que contém metadados.
  • Body - corpo do elemento HTML que contém o conteúdo mostrado no navegador.

Tag's e Atributos

Voltar ao início

Nas aulas "Falando sobre Tag's" e "Atributos das Tags" foram trabalhadas suas formas de funcionamento, tendo por diferença o fato da tag alterar a forma como um elemento HTML aparece na página, enquanto o atributo descreve as características desse elemento:

Tag:

Uma estrutura de linguagem de marcação que contém instruções e possui uma marca de início e outra de fim. As tags são usadas para delimitar estilo e conteúdo.

Atributo:

Uma informação adicional que é passada para a tag para que ela se comporte de forma esperada. Os atributos possuem nome e valor, e podem ser globais ou específicos.

Listas Ordenadas e Não Ordenadas

Voltar ao início

Nessa aula foram trabalhadas listas ordenadas e a lista não-ordenadas, cuja principal diferença está na maneira com que os itens de cada lista são apresentados - a lista ordenada enumera os elementos; a lista não-ordenada coloca marcadores antes de cada elemento.
